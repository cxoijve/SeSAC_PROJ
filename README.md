# 얌얌푸드 SAP ERP 시스템 FI · MM · SD 모듈 개발 및 구축 프로젝트

## 프로젝트 개요
SAP SeSAC 5기의 6조 교육생들이 가상의 밀키트 제조·유통 기업 ‘얌얌푸드’를 대상으로 ERP 시스템을 직접 설계 및 개발한 프로젝트입니다. 구매/자재관리(MM), 판매(SD), 재무회계(FI) 프로세스를 중심으로 통합 프로세스를 구현하였으며, 각 모듈 간의 데이터 흐름을 ABAP 프로그램으로 유기적으로 연계하여 실제 업무 환경과 유사한 시스템이 될 수 있도록 하였습니다.

## 프로젝트 기간
- 시작일: 2025년 9월 2일
- 종료일: 2025년 11월 20일

## 배경 및 문제점
- 복잡한 재고관리, 주문처리, 유통 과정에서 발생하는 비효율성 존재
- 빠르게 변하는 식품 수요에 유연히 대응하는데 어려움

## 해결방안
- SAP S/4HANA ERP 시스템 도입
  - 구매(MM), 판매(SD), 회계(FI) 모듈 통합으로 업무 프로세스 최적화
  - 유통경로 및 공급망 관리 강화
  - 운영비용 절감 및 수익성 향상
